# 🤖 Basic Chatbot using Python

## 📌 Project Overview

The **Basic Chatbot** is a simple rule-based conversational chatbot developed using Python as part of the **CodeAlpha Python Programming Internship**. The chatbot interacts with users by responding to predefined inputs such as greetings, basic questions, and farewell messages.

This project demonstrates the fundamental concepts of Python programming and provides an introduction to building interactive applications using conditional statements and loops.

---

## 🎯 Objective

The main objective of this project is to build a chatbot that can:

* Accept input from the user.
* Recognize predefined commands or questions.
* Respond with appropriate predefined replies.
* Continue the conversation until the user chooses to exit.

---

## ✨ Features

* Responds to greetings such as **"hello"** and **"hi"**.
* Answers simple questions like **"how are you?"** and **"what is your name?"**.
* Handles appreciation messages such as **"thank you"**.
* Ends the conversation when the user types **"bye"**.
* Displays a default message for unknown inputs.
* User-friendly and easy to understand.

## 💻 How It Works

1. The chatbot starts and displays a welcome message.
2. The user enters a message.
3. The program converts the input into lowercase for easy comparison.
4. Based on the user's input, the chatbot provides predefined responses.
5. If the input is not recognized, the chatbot asks the user to try another question.
6. The conversation continues until the user types **"bye"**.

You: hello

## 🚀 How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project in Visual Studio Code or any Python IDE.
4. Run the Python file using the following command:

python basic_chatbot.py

5. Start chatting with the bot through the terminal.
